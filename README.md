# 📝 Task Manager Web App

A modern task management web application built using **HTML, CSS, and JavaScript**.  
This app allows users to manage tasks efficiently with features like status tracking, time tracking, and real-time updates.

---

## 🌐 Live Demo
👉 https://pham0342.github.io/To-Do-List-App/

---

## 🚀 Features

- ✅ Create tasks with title and description  
- 📅 Set due dates for tasks  
- 🔄 Track task status:
  - Pending  
  - In Progress  
  - Completed  
- ⏱️ Record start time and finish time  
- 📊 Automatically calculate time taken to complete tasks  
- ✏️ Edit tasks  
- ❌ Delete tasks  
- 🔍 Filter tasks by status  
- 🌙 Dark mode toggle  
- 💾 Persistent data using **localStorage**  
- ⚡ Real-time UI updates  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Browser LocalStorage
